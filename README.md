# Telegram Line Sticker Bot

協助搬運 Line 貼圖的 Telegram Bot（僅個人學術用途ＱＡＱ）

## 這能做什麼？

目前僅實作靜態貼圖搬運，指令為 `/static 貼圖ID`

## How To 架

請準備一份 config.json 參考 config.example.json 並放置於專案根目錄

目前採取 polling 方式，所以只需要 node 環境（個人是 v12）

``` sh
npm i
npm run start
```

就可以架起屬於你的搬運工ㄌ！
